1. Create Dialogflow Project
    • Dialogflow project must have an intent with the training phrase "TRIGGER AVAYA CLOUD VOICE" to work with voice to begin the convo
    • For any intents which you wish to end the conversation, add the output context 'avaya_cloud_end_conversation' to the intent
  2. Go through authorization steps for Google Cloud
    • Create Service Account
    • Assign Dialogflow API Admin Role
    • Generate JSON key file and download it
    • Record the project-id
  3. Procure a number from Avaya Cloud
  4. Set the Request URLs for Avaya Cloud to:
    • Voice: https://appdev.collaboratory.avaya.com/Dialogflow/cpaas-voice.php
    • SMS: https://appdev.collaboratory.avaya.com/Dialogflow/cpaas.php
